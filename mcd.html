  1. (2 punts) Crea el fitxer mcd.js amb una funció recursiva que calculi el màxim comú divisor de dos nombres amb una funció recursiva seguint el següent algorisme:

    mcd(x, y) és

    • x si y=0

    • el resultat de calcular el mcd de y i el reste de la divisió sencera de x entre y

    • Exporta aquesta funció.

      •                 
let mcd = (x, y) => {
    if (y === 0) return x;
    return mcd(y, x % y);
};
export {mcd};
